# Smart Garage 🚗

## Park your car with no worries

Smart Garage is a modern parking management web project built using HTML, CSS, JavaScript, and PHP.

The project is open-source and welcomes contributions from developers, designers, and students through GitHub Pull Requests.

## Pages

- Home
- About
- Services
- Parking
- Contact
- Login
- Register

## Tech Stack

- HTML5
- CSS3
- JavaScript
- PHP sample contact file

## Notes

GitHub Pages runs the HTML, CSS, and JavaScript files. The `contact.php` file is included as a backend sample for PHP hosting environments.
